# PDF Drive

Простое приложение для **сборки нескольких файлов в один PDF**. Часть набора Senior Hub.

Например, вы прошли медицинское обследование и получили PDF-результаты от разных врачей, сканы анализов в виде картинок и текстовые заметки — PDF Drive соберёт всё это в один аккуратный PDF-файл в нужном вам порядке.

## Возможности

- 📄 Объединение нескольких **PDF**-файлов в один.
- 🖼 Добавление **изображений** (JPG, JPEG, PNG, BMP, GIF, WEBP) — каждое становится страницей PDF.
- 📝 Добавление **текстовых** файлов (TXT, LOG, MD) — текст переносится по словам и разбивается на страницы.
- 📘 Добавление документов **Word** (DOCX) — абзацы (с жирным/курсивом и заголовками), таблицы и встроенные изображения. Сложное оформление Word упрощается.
- Единый список: добавляйте файлы любого типа вперемешку, перетаскивайте мышью или кнопкой «Добавить файлы».
- Изменение порядка стрелками ▲ ▼ и удаление ✕.
- Светлая и тёмная темы, языки: русский / English. Настройки запоминаются.

## Установка

Скачайте `PdfDrive.exe` из раздела [Releases](../../releases) и запустите — установка не требуется (self-contained, .NET в комплекте, Windows x64).

## Сборка из исходников

```bash
dotnet build -c Release
```

Требуется .NET SDK 9.0+. Релизный единый exe:

```bash
dotnet publish -c Release
```

## Технологии
